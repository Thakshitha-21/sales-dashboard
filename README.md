 Sales Dashboard

A modern sales analytics dashboard built with Next.js 15, TypeScript, and Tailwind CSS. Visualize and analyze sales performance across multiple years with interactive charts and real-time filtering.

 Features

- Interactive Charts - Bar, Line, and Pie charts using Recharts
- Year Filtering - Switch between 2022, 2023, and 2024 sales data
- Custom Threshold Filter - Filter data by minimum sales value
- Responsive Design - Works on desktop, tablet, and mobile
- Real-time Updates - Charts update instantly on filter changes
- Type Safety - Built with TypeScript for better development experience

 Tech Stack

- Framework: Next.js 15 with App Router
- Language: TypeScript
- Styling: Tailwind CSS
- Charts: Recharts
- Icons: Lucide React
